# .github

This repo carries the **org-level profile README** that GitHub shows above the repo list on
[github.com/CasualOffice](https://github.com/CasualOffice) — the file lives at
[`profile/README.md`](profile/README.md).

It can also hold:

- `.github/workflows/*.yml` — workflows reused across the org
- `.github/ISSUE_TEMPLATE/*` — default issue templates inherited by every repo without its own
- `.github/PULL_REQUEST_TEMPLATE.md` — default PR template
- `.github/FUNDING.yml` — sponsorship buttons

Edit `profile/README.md` to update the front page of the organisation.
